# v0.1.0

Initial public prototype release of **OpenClaw Auto Heal**.

## Highlights

- Bootstrapped the repository and project structure
- Added bilingual documentation entry points (`README.md` + `README.zh-CN.md`)
- Implemented layered recovery strategy:
  - safe-backup restore
  - `openclaw doctor --fix`
  - AI fallback repair
- Added standalone AI config support with OpenClaw config fallback
- Added safety validation for AI-generated repair scripts
- Added dry-run and diff output support
- Added installer scripts for easier local setup
- Added architecture and security documentation
- Added GitHub Actions CI for basic validation and shell syntax checks

## Positioning

This release is intended as an **experimental public prototype**.
It is suitable for public iteration, early adopters, and feedback collection — not yet as a production-grade hardened release.
