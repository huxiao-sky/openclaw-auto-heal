# Release Announcement Draft

## Title
OpenClaw Auto Heal v0.1.0 — initial public prototype

## Body
OpenClaw Auto Heal is now public.

This project is a lightweight self-healing runtime protection layer for OpenClaw Gateway. It combines launchd supervision, health checks, official doctor-first recovery, and AI fallback repair into one practical workflow.

### Highlights
- doctor-first repair via `openclaw doctor --fix`
- AI fallback with safety validation
- standalone AI credential support
- dry-run and diff support
- installer script
- bilingual README
