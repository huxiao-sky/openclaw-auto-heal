# Launch Post Draft

Built a new open-source project today: **OpenClaw Auto Heal**.

It adds a practical self-healing layer on top of OpenClaw Gateway:
- safe-backup recovery
- doctor-first repair with `openclaw doctor --fix`
- AI fallback repair with safety guards
- dry-run and diff support

The goal is simple: when Gateway breaks, don’t just alert — recover.
