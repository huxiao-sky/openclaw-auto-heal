# GitHub Release Template

## Title
v0.1.0 — Initial public prototype

## Summary
OpenClaw Auto Heal is a self-healing runtime protection layer for OpenClaw Gateway.
It combines launchd supervision, health checks, official doctor-first recovery, and AI fallback repair into a lightweight recovery loop.

## Highlights
- doctor-first recovery via `openclaw doctor --fix`
- AI fallback repair with safety guards
- standalone AI credentials supported
- dry-run and diff support
- bilingual README
- installer scripts

## Notes
This is an experimental early-stage open-source release.
Use it as a public prototype, not as a fully production-hardened operations platform.
